==================

#           Diese Version ist die DLL von Maximus 3.0 und behebt einige

a) Machen Sie Sicherheitskopien der *.INI-Dateien im FleetStreet-Verzeichnis.

e) Installationsprogramm beenden.

Wenn Sie FleetStreet zum ersten Mal installiert haben:

f) Bedienungsanleitung lesen!
g) FleetStreet starten. Das Setup-Notebook und die Onlinehilfe îffnen sich
   nach einer Meldung von selbst.

i) Auf die "Tosser"-Seite wechseln.
j) Verwendeten Tosser aus der Liste auswÑhlen. Tosser-Konfiguration einlesen.
   Dabei werden die Netz-Adressen und die Origin-Zeile miteingelesen. Evtl. auf
   der "Adressen"-Seite nochmal kontrollieren.
